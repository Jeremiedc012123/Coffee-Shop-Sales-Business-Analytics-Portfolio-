# Coffee-Shop-Sales-Business-Analytics-Portfolio-
A comprehensive business intelligence and sales analytics project built using the Maven Analytics Coffee Shop Sales dataset. This project focuses on transforming raw transactional data into actionable business insights through data cleaning, exploratory data analysis, KPI monitoring, and interactive dashboard visualization.

The dataset contains over 149,000 transaction records from Maven Roasters, a fictional coffee shop chain operating across multiple locations in New York City. It includes detailed transaction timestamps, store locations, product categories, pricing, and sales quantities.
